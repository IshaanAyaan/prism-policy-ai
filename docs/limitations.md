# Limitations

- The v3 text layer is not based exclusively on official statute text. Coverage provenance in the mechanism state-year artifact includes substantial supplemental text.
- The causal audit is built around a small number of clean beer-tax events, which constrains statistical power.
- The scenario outputs are model-based forecasts and should not be read as policy-effect estimates.
- This public repo ships frozen benchmark and summary artifacts. It does not attempt to recreate the original raw-data ingestion or every experiment from the larger workspace.
- Teen-outcome analysis is intentionally out of scope here. It is archived in V2 rather than exposed as an active public module.
